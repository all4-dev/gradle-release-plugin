# Gradle Release Plugin

A Gradle plugin for streamlined library publishing with built-in support for Maven Local, Maven Central, GitHub Packages, and GitHub Pages. Features include library groups for multi-module projects, dry-run mode for safe testing, per-library versioning, and automatic changelog integration for GitHub releases.
